# NotesApp
Implement a Notes App using FileManager

*This is simple notes app using UserDefaults*

## Elements

* UIViewController
* UITextField
* UITextView
* UITableView
* UILabel
* UITableViewDelegate
* UITableViewDataSource
* FileManager
